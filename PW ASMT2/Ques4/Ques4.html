Ordered List with Letters

  1. div Tag
    2. The div element is often used as a container for other HTML elements. Defines a section in a document (block-level)
  2. a Tags
    3. The tag defines a hyperlink, which is used to link from one page to another. The most important attribute of the element is the href attribute, which indicates the link's destination. By default, links will appear as follows in all browsers:
    An unvisited link is underlined and blue
    A visited link is underlined and purple
    An active link is underlined and red
  3. Embed Tags

    4. The tag defines a container for an external resource, such as a web page, a picture, a media player, or a plug-in application.
